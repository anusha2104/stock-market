# Stock Market Analyzer 📊

An AI-based stock market analysis system that predicts stock prices using financial indicators.

## Features
- Real-time stock data
- Indian stock support (.NS, .BO)
- Technical indicators (SMA, EMA, Volatility)
- Prediction engine
- Trend analysis

## Tech Stack
- Frontend: HTML, CSS, JS
- Backend: Flask
- Data: Yahoo Finance API

## Run Instructions

cd backend  
pip install -r requirements.txt  
python app.py  

Open frontend/index.html
